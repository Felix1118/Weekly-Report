# DFBN 文献阅读工作总结（上周）

## 1）我做了什么
- **完成 4 篇 DFBN（动态功能脑网络）方向论文精读与要点摘录**，聚焦三条主线：**时空耦合建模 + 拓扑演化 + 可解释性**。整体感受是：不少既有方法仍倾向把**空间拓扑**与**时间依赖**拆开提取，导致时空相关难以有效耦合，同时可解释性也偏弱。
- **梳理并对比了代表性建模范式与关键模块：**
  - **长间隔时序交互建模**：用 **Hawkes 过程**刻画跨窗口脑区间的长间隔信息传递，并用交互式图卷积框架协同抽取时空特征；另外引入**基于超图的空间信息增强（SIA）**以捕捉更复杂的高阶依赖。
  - **可解释分层 DFBN 表征（功能/结构先验）**：基于功能与结构聚类，把脑网络划分为多个功能子网络（如 **DMN、FPN、SAN** 等），通过 **intra-graph + inter-graph** 传播协同提取子网络内部与跨窗口的动态特征，并配合子网络约束与注意力融合提升可解释性。
  - **拓扑演化图学习（TEG）**：将相邻窗口的 **hubness** 分布视为源/目标域，用 **Wasserstein 距离（WD）**与 **Gromov-Wasserstein（GWD）**学习节点/边层面的拓扑演化过程，并用 **PRI** 引导结构保留“最相关、最少冗余”的信息；最后用**多跳 GCN** 联合抽取长程空间相关与动态时间特征。
  - **Hubness 传播 + 最优传输（OT）**：强调不仅要刻画动态演化，还要抓住高阶信息传播过程；并指出采用 **OT** 来模拟跨窗口演化、结合带时空注意力的模型可自动聚焦更有价值的时空信息，从而获得更好的分类表现。

## 2）存在的问题
- 目前对这些方法的理解还主要停留在**论文思路层面**，对**网络具体怎么实现**、代码里**每一步怎么走**还不够清楚，需要通过复现把整体流程跑通。
- 一些关键环节在论文里描述偏抽象，落地到实现会卡点，比如：
  - **DFBN 如何构建**：滑窗怎么设、相关性怎么算、阈值/稀疏化怎么做、节点/边特征怎么定义等；
  - **时空模块如何对接**：时间建模（跨窗依赖/演化）和图卷积（空间拓扑）在代码中怎么串联、输入输出张量格式是什么；
  - **训练/评估口径差异**：窗口长度、步长、数据划分、正则与损失等差异会导致复现结果波动，缺少统一口径时很难公平比较。
